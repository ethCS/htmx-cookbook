# CSCI322 Cookbook

A recipe discovery and personal cookbook app built with React and Firebase.

**Best viewed on desktop.**

## Live Site
https://csciumt.web.app

## GitHub
https://github.com/ethCS/cookbook

## Features
- Browse and search thousands of recipes via TheMealDB API
- Save favorite recipes to your personal collection
- Create and manage your own custom recipes
- User authentication with Firebase Auth

## Tech Stack
- React + Vite
- Firebase (Auth + Firestore + Hosting)
- Tailwind CSS
- TheMealDB API
